# ensure-slash

> Ensure your string ends with a slash

## Installation

```
$ npm install ensure-slash
```

## Usage

```js
const ensureSlash = require('ensure-slash')

ensureSlash('/home/foo')
// => '/home/foo/'

ensureSlash('/home/bar/')
// => '/home/bar/'
```
